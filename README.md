# RaftaarRide

Premium ride-hailing platform built with Expo (React Native) + React Admin Panel + Firebase + Razorpay.

## Features
- Dark UI with glass card design
- Hindi Voice AI assistant
- Phone OTP + Email authentication (Firebase)
- Razorpay LIVE payments
- 14-17% cheaper than Rapido
- 6.7% admin commission
- Full Hindi/English + Dark/Light toggle

## Apps
- **Mobile App**: Expo (React Native) — passenger & driver apps
- **Admin Panel**: React + Vite — deployed on Vercel
- **API Server**: Express + PostgreSQL backend

## Tech Stack
- Firebase Auth + Firestore
- Razorpay (LIVE mode)
- PostgreSQL (Drizzle ORM)
- Expo Router
- React + Vite (Admin)

Vercel redeploy trigger after environment variable update
