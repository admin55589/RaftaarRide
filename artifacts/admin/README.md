# RaftaarRide — Admin Panel (React + Vite)

## Setup

```bash
pnpm install
pnpm run dev
```

## Default Login
- Email: `admin@raftaarride.com`
- Password: `Admin@123`

## Features
- Dashboard with revenue analytics
- KYC approval / rejection
- Driver management
- User management
- Rides history
- Withdrawal requests management

## Build for Production
```bash
pnpm run build
```
